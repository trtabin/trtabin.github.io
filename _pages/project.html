IoT-core: A Cloud based IoT Software Platform [2023]

• It’s a cloud service which can store and visualize data collected from thousands of IoT gateways and app.
• New devices are automatically added to the platform by a proper authentication system.
• Users can send various commands and update the device from the cloud platform.
• It can generate events to notify the user when a predefined condition matches.
