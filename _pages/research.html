Development of an Sensor Network Communication Protocol [Jan.2024 – Apr. 2024]

• Developed an application-layer protocol that enables automatic registration and integration of new sensor nodes for simplified network deployment over the CAN bus.
• Designed a scalable architecture to support a growing number of sensor.
• Implemented a packet reordering mechanism to address out-of-sequence delivery.
• Achieved a 40% increase in data acquisition speed compared to the previous manually-configured system by streamlining the node integration process.

Comparative Study on Feature Selection for Heart Disease Prediction [Dec.2021 – Nov. 2022]

• Different feature selection techniques such as filter, wrapper, and embedded methods are used for heart disease prediction.
• Compare the performance of feature selection techniques with different classification algorithms, such as logistic regression, decision trees, support vector machines, and deep learning algorithms, in the context of heart disease prediction.
